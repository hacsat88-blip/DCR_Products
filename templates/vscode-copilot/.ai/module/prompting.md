# Prompting Module

Use this module when the user asks to improve, rewrite, refine, or systematize a prompt.

Process:
1. identify the real goal
2. identify ambiguity and missing constraints
3. improve structure and clarity
4. define output format
5. produce the refined prompt

Output format:
- Analysis
- Improvement Points
- Refined Prompt

Principles:
- prefer explicit roles
- prefer concrete instructions
- define constraints clearly
- define the expected output shape
- remove vague wording
- keep the prompt reusable
